# Portfolio Website

Статический сайт-портфолио с профессиональной главной страницей, раскрывающимися разделами и дополнительными вкладками.

## Как открыть

Откройте `index.html` в браузере или запустите локальный сервер:

```powershell
node server.js
```

После запуска сайт будет доступен по адресу `http://localhost:4173`.

## Что заменить под себя

- `assets/profile-placeholder.svg` на реальную фотографию профиля.
- `Ваша текущая должность` на актуальный пост.
- Тексты в разделах `Обо мне`, `Опыт`, `Проекты`, `Навыки`.
- Контакты в блоке `Контакты`.
